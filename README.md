## Getting Started

First, install dependencies:

```bash
pnpm i
```

Then start development server via:

```bash
pnpm dev
```

If you want to use docker:

```bash
docker compose up -d --build
```

Project can be found at: 

https://weather-app-ebon-rho.vercel.app/
